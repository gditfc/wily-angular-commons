Date Picker
The date picker is a component used to select a date. It includes validation on the input, and a calendar so users can select a date through a more friendly interface.

The look of the calendar input can be customized to fit into any form, through custom classes.
Dialog
The dialog can be used to open a window over the page content to perform sub-functions or display warnings or important information to the user.

The dialog is fully accessible and responsive, and supports features such as opening multiple dialogs on top of each other, trapping focus and close on escape.
Dropdown
The dropdown component is a customizable component based on the default HTML 'Select'. It was made to behave similarly to the native select, and is fully accessible.

The look of the dropdown can be customized to fit into any form, through custom classes.

Icon Select
The icon select is a dialog that allows a user to choose an icon from a collection of fontawesome and custom wily icons (shown below). The dialog supports filtering on icons.
Paginator
The paginator is a component that makes it easy to split data into multiple 'pages.' The paginator includes multiple versions with different looks and supported functions.

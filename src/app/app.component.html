The date picker is a component used to select a date. It includes validation on the input, and a calendar so users can select a date through a more friendly interface.

Dialog
The dialog can be used to open a window over the page content to perform sub-functions or display warnings or important information to the user.

The dialog is fully accessible and responsive, and supports features such as opening multiple dialogs on top of each other, trapping focus and close on escape.
